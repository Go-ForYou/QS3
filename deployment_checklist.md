## 📋 部署步骤

### 第一步：创建服务
- [ ] 在Render中创建新Web Service
- [ ] 连接GitHub仓库
- [ ] 选择Python环境

### 第二步：配置服务
- [ ] 设置服务名称
- [ ] 配置构建命令：`pip install -r requirements.txt`
- [ ] 配置启动命令：`gunicorn app:app --bind 0.0.0.0:$PORT`
- [ ] 选择免费计划

### 第三步：环境变量
- [ ] `SECRET_KEY` 自动生成
- [ ] `FLASK_ENV` 设置为 production
- [ ] `PORT` 自动设置

### 第四步：部署
- [ ] 点击创建服务
- [ ] 等待构建完成
- [ ] 确认状态为 "Live"
